File structure:

setup
	The files for the setup dll which interfaces with the com0com driver
	
setupc
	The command line utility that ueses the dll to allow command line access to the driver

include
	Additional files which where also used when compiling the drivers

driver_bin_x64
	The 64 bit com0com driver binaries extracted from the original installers from sourceforge
	
driver_bin_x86
	The 32 bit com0com driver binaries extracted from the original installers from sourceforge
